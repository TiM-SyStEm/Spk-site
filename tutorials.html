Основы работы с переменными и вводом.

Переменная - это ячейка памяти компьютера, которая имеет имя и хранит некоторое значение. Так же переменные могут меняться во время выполнении программы. При записи в ячейку нового значения старое стирается.

Объявление переменной:
var имя = значение

Чтобы ввести переменную, необходимо написать: var имя = input : “ ”

Рассмотрим пример, напишем простенький калькулятор сложения:

var a = input : "Введите первое число: "
var b = input : "Введите второе число: "
var c = toInt(a) + toInt(b) out : c

Конвертирирование переменных в различные типы данных.

На данный момент в SPK существуют 3 типа данных. А именно: Int - целочисленный тип данных. Float - тип данных для чисел с плавающей точкой. Str(String) - строковой тип данных.

Чтобы конвертировать в тот или иной тип данных нужно написать toInt(значение) или toStr(значение) или toFloat(значение).

Когда мы писали калькулятор сложения, то мы уже конвертировали тип данных String в тип данных Int, когда мы вводим число в переменной тип данных автоматически становится Str, если мы не будем конвертировать результат, то строки будут складываться, например возьмем числа 7 и 5, их надо сложить, что будет если мы не будем конвертировать в тип Int? Тогда результат будет не 12, а 75.

Примечание: Следует использовать конвертирование, т.к если вы в условии (об этом чуть позже) не конвертируете переменную , то программа не будет работать!

Условия. Оператор if - неполная форма.

Условный оператор – это конструкция языка программирования, позволяющая выполнять различные наборы блоков кода в зависимости от результата вычисления логического выражения

Синтаксис:
if (лог. выражение) {
действие1
действие2
...
действиеN
}

Если условие выполняется, выполнится действия , которое указано в квадратных скобках. А если он не выполнится, то... (информация в следущем блоке)

Напишем программу которая меняет числа, если a < b:

var a = input : "Введите первое число: "
var b = input : "Введите второе число: "
if (toInt(a) + toInt(b)) {
var tmp = a
a = b
b = tmp
}
out : a
out : b

Примечание: Условие содержит всегда два исходных результата, или оно верное(True) или оно ложное (False).

Циклы.

Циклы - это основные управляющие конструкции в программировании. С помощью циклов в программировании организуется многократное выполнение одних и тех же участков кода.

Циклы бывают в постусловием (do) и с предусловием (while, for) Цикл с постусловием — цикл, в котором условие проверяется после выполнения тела цикла. Отсюда следует, что тело всегда выполняется хотя бы один раз. Цикл с предусловием — цикл, который выполняется, пока истинно некоторое условие, указанное перед его началом. Это условие проверяется до выполнения тела цикла, поэтому тело может быть не выполнено ни разу (если условие с самого начала ложно).

Начнем с цикла for

Синтаксиc:
for (поле1 поле2 поле3) {
тело цикла
}

Поле1: инцилизируем переменную Поле2: помещаем в поле3 (обычно это диапозон) Поле3: диапозон значений, которое будет повторяться столько раз, сколько задан диапозон чисел!

Пример:

for x in 0,7 {
out : "Hello, world!"
}

Цикл while

Синтаксиc:
while (условие) {
тело цикла
}

Пример:

var i = 1
while (i <= 10) {
out : "Hello, world!"
i = i + 1
}

Цикл do

Операторы stop и continue stop – это оператор принудительного выхода из циклов while, do, for. continue – оператор принудительного выхода из текущей итерации цикл

Синтаксиc:
do {
действие1
действие2
...
действиеN
}

Пример:

do {
var inp = input : ""
inp = toInt(inp)
if (inp < 5) {
stop
}
else if (inp == 5) {
out : "Итерация прервана!"
continue
}
}

Функции

Функция - это независимая именованная часть программы, которую после однократного описания можно многократно вызвать по имени из последующих частей программы.

Синтаксис:
fun имя_функции (параметры) { тело функции

ВНИМАНИЕ: Функцию нужно вызывать, в противном случае программа не выполнится

Пример:

fun test(n, k) {
var result = toInt(n) + toInt(k)
out : result
}
var a = input : "Введите первое число: "
var b = input : "Введите второе число: "
test(a, b)

Оператор return – возвращает результ.

Пример:

fun test() {
out : "I love SPK!!!"
return 0
}
test()

После выполнения оператора return, происходит выход из функции.

Классы

Класс - это абстрактный тип данных. С помощью которых описывается некоторая сущность (хар-ка и возможные действия). Кстати, класс нельзя вызывать, его можно только указывать.

Синтаксис:
class имя_класса {
тело класса
}

Пример:

class Test {
var a = 0
fun test1() {
out : 5
}
fun test2() {
pass
}
fun test3() {
pass
}
}
Test.test1()
out : Test.a

Если мы создаем функцию, которая не хранит в себе код, то вызываем оператор pass.

Поговорим о модификаторах доступа.

Модификаторы доступа позволяют задать допустимую область видимости для членов класса. То есть модификаторы доступа определяют контекст, в котором можно употреблять данную переменную

Изначально все классы являются публичными, то есть все что имеется за телом класса.

На данный момент в SPK всего два модификатора: protected - Доступ ограничен содержащим классом или типами, которые являются производными от содержащего класса.

private - Доступ ограничен содержащим типом. Он запрещает обращаться к переменным или функциям класса. Доступ ограничен если функция или переменная вызывается за классом.

Например:

protected:
class Class1 {
fun printOne() {
out : 1
}
out : Class1.printOne()
