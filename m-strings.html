Strings


This module will help you work with strings
special-pm install strings

Importing

The Strings module can be imported as well as all library libraries, namely: Add strings

Information of classes

There are 2 classes in the Strings module. String and Character. Character - for working with characters, String - for working with strings.

Each of them has its own methods that simplify working with strings and characters.

Class String

charAt(string: str, index: int) - returns a character at the index in string
indexOf(string: str, chars: str)returns the index of the first occurrence of chars in string
equals(string1: str, string2: str) - a function that returns the correspondence of two strings
contains(string1: str, string2) - (analog of string1 in string2) a function that returns the presence of string2 in string1

Class Character

isWhitespace(char: str) - returns char matching to a space
isDigit(char: str) - returns a char match to a digit
isLetter(char: str) - returns a char match to a letter
whitespace() - returns a space " "
